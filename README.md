# Tradixa Retail Management System

## About
Tradixa adalah sistem manajemen retail dan ERP lengkap untuk optimalisasi operasional bisnis.

## Getting Started

1. Clone repository
2. Install dependencies: `npm install`
3. Buat file `.env.local` dan set environment variables yang sesuai.

4. Jalankan: `npm run dev`

## Features
- Inventory Management
- Sales Transaction & Invoice
- Purchase Orders & Goods Receipt
- Financial Management (Payables & Receivables)
- Bank Account Management
- Financial Agent (PPOB)
- HR Management
- Customer CRM & Loyalty Program
- Marketing Automation
- Discount Management
- Reports & Analytics
- AI Assistant (Tradixa Assistant)
